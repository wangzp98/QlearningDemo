# 基于Q-learning的最短路径吃糖果



程序结构：

- data.py：关于程序初始化的数据

- myMap.py:地图绘制代码

- myMap.ui:地图源文件，可用于生成python文件

- myOperationMain.py:程序入口文件，包含操作地图刷新与Qlearning的主循环代码

- QlearningMain.py: Qlearning的辅助函数



环境：

- 系统：Windows10、MacOS
- 软件：PyCharm、Qt Designer
- 编程语言：Python3.6.8
- 所需包：numpy、pandas、PyQt5、PyQt5-tools
